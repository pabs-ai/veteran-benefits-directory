# State Veterans Benefits Directory

A polished, mobile-first single-page directory that helps veterans find state-level benefits and resources quickly. This repo contains a static HTML guide built for accessibility, responsiveness, and easy navigation.

## What it is

- A searchable, state-based veterans benefits directory.
- Designed as a fast, standalone static page with modern layout and visual hierarchy.
- Ideal for distributing a lightweight reference to veterans, families, and support organizations.

## Highlights

- ✅ Responsive design for desktop, tablet, and mobile
- ✅ Clean UI with bold, patriotic styling
- ✅ Easy to host as a static site
- ✅ No dependencies — just plain HTML/CSS
- ✅ Built for clarity and readability

## Files

- `state-veterans-benefits-directory.html` — main page and interface

## How to use

1. Open `state-veterans-benefits-directory.html` in your browser.
2. Review the state benefits listings.
3. Share the file or host it on GitHub Pages for public access.

## Why it matters

This project supports veterans by making benefit information easier to access without complicated navigation or external scripts. It’s best used as a shareable reference or an entry point for an expanded veterans support portal.
